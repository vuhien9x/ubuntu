Cài đặt Node.js trên Ubuntu

Hướng dẫn chi tiết cách cài đặt Node.js và npm (Node Package Manager) trên Ubuntu.

Bước 1: Cập nhật hệ thống

Đảm bảo hệ thống của bạn được cập nhật trước khi cài đặt bất kỳ phần mềm mới nào.

  1. Mở Terminal (Ctrl + Alt + T).
  2. Cập nhật danh sách gói: 
    sudo apt update
  3. Nâng cấp các gói đã cài đặt: 
    sudo apt upgrade -y

Bước 2: Cài đặt Node.js

Bạn có thể cài đặt Node.js từ kho lưu trữ mặc định của Ubuntu hoặc từ NodeSource. NodeSource thường cung cấp các phiên bản Node.js mới hơn.

Cách 1: Cài đặt từ kho Ubuntu

Cách này đơn giản nhưng có thể không phải là phiên bản mới nhất.

  1. Cài đặt Node.js: 
    sudo apt install nodejs -y
  2. Cài đặt npm (Node Package Manager): 
    sudo apt install npm -y

Cách 2: Cài đặt từ NodeSource (Khuyến nghị)

Cách này cho phép bạn cài đặt các phiên bản Node.js cụ thể và thường là phiên bản mới nhất.

  1. Cài đặt curl (nếu chưa có): 
    sudo apt install curl -y
  2. Thêm kho lưu trữ NodeSource. Ví dụ, để cài đặt Node.js 18.x: 
    curl -sL https://deb.nodesource.com/setup_18.x | sudo -E bash -

    (Thay "18.x" bằng phiên bản bạn muốn, ví dụ "20.x", "16.x", v.v.)

  3. Cài đặt Node.js: 
    sudo apt install nodejs -y

    (Lệnh này cũng sẽ cài đặt npm)

Bước 3: Kiểm tra cài đặt

Xác minh rằng Node.js và npm đã được cài đặt thành công.

  1. Kiểm tra phiên bản Node.js: 
    node -v
  2. Kiểm tra phiên bản npm: 
    npm -v

Kết quả

Bạn đã cài đặt thành công Node.js và npm trên Ubuntu! Bây giờ bạn có thể bắt đầu phát triển các ứng dụng JavaScript phía máy chủ.
