Cài đặt Laravel Sail và Rút gọn lệnh

Hướng dẫn cài đặt Laravel Sail và cách rút gọn lệnh ./vendor/bin/sail cho tiện sử dụng.

1. Cài đặt Laravel Sail

Để cài đặt Laravel Sail, bạn có thể sử dụng Composer. 

composer require laravel/sail --dev

2. Khởi tạo Sail

Sau khi cài đặt, bạn cần khởi tạo Sail bằng lệnh Artisan. 

php artisan sail:install

Hướng dẫn rút gọn lệnh Sail

Để thuận tiện hơn khi sử dụng, bạn có thể rút gọn lệnh ./vendor/bin/sail thành sail.

1. Xác định shell đang sử dụng

Trước tiên, bạn cần biết mình đang dùng shell nào (Bash hoặc Zsh). 

echo $SHELL
  • Nếu kết quả là /bin/bash, bạn đang dùng Bash.
  • Nếu kết quả là /bin/zsh, bạn đang dùng Zsh (thường là trên macOS).

2. Thêm alias vào file cấu hình shell

Tiếp theo, bạn sẽ thêm một alias vào file cấu hình của shell.

  • Với Bash, file cấu hình thường là ~/.bashrc hoặc ~/.bash_profile.
  • Với Zsh, file cấu hình là ~/.zshrc.

Bạn có thể mở file này bằng trình soạn thảo nano. 

nano ~/.bashrc

(Hoặc ~/.bash_profile, ~/.zshrc tùy vào shell bạn đang dùng).

3. Thêm alias

Thêm một trong các dòng sau vào cuối file cấu hình. Khuyến nghị nên dùng `alias` hoặc `$PATH`. 

alias sail='./vendor/bin/sail'

Hoặc 

export PATH="$PATH:./vendor/bin"

Hoặc 

function sail { .\vendor\bin\sail $args }

Lưu file và thoát (Ctrl + O, Enter, Ctrl + X nếu dùng nano).

4. Áp dụng thay đổi

Để các thay đổi có hiệu lực, bạn cần tải lại file cấu hình shell. 

source ~/.bashrc

(Hoặc ~/.bash_profile, ~/.zshrc tương ứng).

5. Kiểm tra

Bây giờ bạn có thể dùng `sail` thay cho `./vendor/bin/sail`.

Ví dụ: 

sail up
sail artisan migrate

Lưu ý: Alias chỉ hoạt động trong thư mục dự án chứa ./vendor/bin/sail. Nếu bạn di chuyển ra ngoài thư mục dự án, lệnh sail sẽ không hoạt động.
