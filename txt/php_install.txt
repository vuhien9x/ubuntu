Hướng dẫn cài đặt PHP trên Ubuntu

1. Cập nhật hệ thống
   - Mở Terminal (Ctrl + Alt + T)
   - Chạy:
     sudo apt update
     sudo apt upgrade -y

2. Cài đặt PHP
   - Cài PHP mặc định (thường là 8.1 hoặc 8.2):
     sudo apt install php -y
   - (Tùy chọn) Cài thêm module phổ biến:
     sudo apt install php-cli php-mbstring php-xml php-mysql php-curl php-zip -y
     + php-cli: Chạy PHP từ Terminal
     + php-mbstring: Xử lý chuỗi đa byte
     + php-xml: Hỗ trợ XML
     + php-mysql: Kết nối MySQL
     + php-curl: Gọi API/tải dữ liệu web
     + php-zip: Làm việc với file nén
   - Kiểm tra phiên bản:
     php -v (Kết quả ví dụ: PHP 8.1.2...)

3. (Tùy chọn) Cài PHP phiên bản cụ thể
   - Thêm PPA:
     sudo add-apt-repository ppa:ondrej/php
     sudo apt update
   - Cài ví dụ PHP 7.4:
     sudo apt install php7.4 php7.4-cli php7.4-mbstring php7.4-xml php7.4-mysql -y
   - Kiểm tra:
     php7.4 -v

4. (Tùy chọn) Cài server web để chạy PHP
   - Apache:
     sudo apt install apache2 libapache2-mod-php -y
     sudo systemctl restart apache2
   - Nginx:
     sudo apt install nginx php-fpm -y
     sudo systemctl restart nginx
     sudo systemctl restart php8.1-fpm (Thay 8.1 bằng phiên bản của bạn)
   - Tạo file thử:
     echo "<?php phpinfo(); ?>" | sudo tee /var/www/html/info.php
   - Kiểm tra: Truy cập http://địa-chỉ-ip/info.php trên trình duyệt

Kết quả:
   - PHP đã cài đặt và sẵn sàng sử dụng.
