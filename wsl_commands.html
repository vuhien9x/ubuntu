Các Lệnh WSL Thường Dùng

Tương tác cơ bản với WSL

wsl
Mở shell mặc định.
wsl <command>
Chạy lệnh Linux từ CMD/PowerShell.
wsl -d <DistributionName>
Mở shell cho bản phân phối cụ thể.
wsl --list
Liệt kê các bản phân phối đã cài đặt.
wsl --list --verbose
Liệt kê chi tiết các bản phân phối.
wsl --setdefault <DistributionName>
Đặt bản phân phối mặc định.
wsl --unregister <DistributionName>
Gỡ cài đặt bản phân phối (xóa dữ liệu!).
wsl --export <DistributionName> <FileName>
Sao lưu bản phân phối vào file tar.
wsl --import <DistributionName> <InstallLocation> <FileName>
Phục hồi bản phân phối từ file tar.
exit
Đóng shell WSL.

Quản lý file và thư mục

cd <path>
Thay đổi thư mục làm việc.
ls [-l -a -h -t]
Liệt kê file và thư mục.
pwd
Hiển thị đường dẫn thư mục hiện tại.
mkdir <directory_name>
Tạo thư mục mới.
rm <file_name> [-r -f]
Xóa file hoặc thư mục.
cp <source> <destination> [-r]
Sao chép file hoặc thư mục.
mv <source> <destination>
Di chuyển hoặc đổi tên.
cat <file_name>
Hiển thị nội dung file.
less <file_name> / more <file_name>
Xem nội dung file theo trang.
nano <file_name> / vim <file_name>
Soạn thảo văn bản.

Quản lý gói (thường dùng với Ubuntu)

sudo apt update
Cập nhật danh sách gói.
sudo apt upgrade
Nâng cấp các gói đã cài đặt.
sudo apt install <package_name>
Cài đặt gói mới.
sudo apt remove <package_name>
Gỡ cài đặt gói (giữ cấu hình).
sudo apt purge <package_name>
Gỡ cài đặt gói (xóa cả cấu hình).
apt search <keyword>
Tìm kiếm gói.
apt show <package_name>
Hiển thị thông tin gói.

Quản lý tiến trình

ps aux
Liệt kê tất cả tiến trình.
top / htop
Giám sát tiến trình và tài nguyên.
kill <PID> / kill -9 <PID>
Kết thúc tiến trình.

Mạng

ip a / ifconfig
Thông tin giao diện mạng.
ping <hostname_or_ip>
Kiểm tra kết nối mạng.
netstat -tulnp / ss -tulnp
Thông tin kết nối mạng.

Quyền và người dùng

chmod <mode> <file_or_directory>
Thay đổi quyền truy cập.
chown <user>:<group> <file_or_directory>
Thay đổi sở hữu.
sudo
Chạy lệnh với quyền root.

Khác

history
Lịch sử lệnh.
clear
Xóa màn hình.
grep <pattern> <file> [-r -i]
Tìm kiếm văn bản.
head <file_name>
Xem đầu file.
tail <file_name> [-f]
Xem cuối file.
